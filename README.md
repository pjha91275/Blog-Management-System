# 🌐 Blog Management System – Full Stack Application

A full-stack blog web application built using **Node.js**, **Express.js**, **MongoDB Atlas**, and **EJS**.  
The project focuses on server-side rendering, authentication, and CRUD-based content management.

---

## 🚀 Features

### 🌍 Frontend
- Built using **HTML**, **CSS**, and **JavaScript**
- Server-side rendered pages using **EJS templating**
- Responsive layout with a custom **Light/Dark theme system**
- Immersive **image carousel** on the homepage
- Modern **glassmorphic card design** with interactive hover effects and focus blurring
- Dedicated admin login entry point on the homepage

### 🔧 Backend
- Developed with **Node.js** and **Express.js**
- Modular routing and middleware-based architecture
- **Markdown parsing** integration for rich blog content support
- Input validation and centralized error handling
- Secure handling of requests and responses

### 🗄️ Database
- Uses **MongoDB Atlas** for cloud-based data storage
- Schema-based data modeling using **Mongoose**
- Validation rules for blog content and users

### 🔐 Authentication
- Admin authentication using **JWT**
- Session management via **HTTP-only cookies**
- Route protection for admin-specific actions

### 📝 Blog Management
- Create, read, update, and delete (CRUD) blog posts
- **Markdown-enabled content** for headings, bold text, and lists
- Automatic timestamps for published content
- Secure storage and retrieval of blog data

---

## 🗺️ Application Architecture & Flow

### 🔹 Public Discovery
- **Home (Feed)**: The central hub featuring an interactive image carousel and a dynamic listing of newest posts with glassmorphic preview cards.
- **Article Depth (Post View)**: A dedicated, focused reading environment that parses raw Markdown into a clean, formatted layout.
- **Identity Portal (About)**: A visual showcase of the platform's vision and the sophisticated technology stack behind it.

### 🔹 Content Management
- **Admin Gateway**: A secure authentication portal for platform administrators.
- **Dashboard Control**: A private interface for managing the entire content repository, allowing for seamless article creation and editing.

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Authentication:** JWT, Cookies
